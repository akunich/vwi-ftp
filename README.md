# VWI FTP - Monsta FTP
Web FTP Client Plugin for Vesta Web Interface

Installation Instructions:

1. Upload the 'ftp' folder to the 'plugins' folder of your VWI installation.
2. Edit the VWI settings and add "ftp" to the plugins list.
